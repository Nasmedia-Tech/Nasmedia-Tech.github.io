# NAP MX SDK 개요

NAP MX SDK는 앱 매체에 광고를 노출하기 위한 미디에이션 SDK입니다. 단일 SDK로 다수의 광고 네트워크를 동시에 운영할 수 있습니다.

## 지원 광고 포맷

| 포맷 | 설명 |
|------|------|
| 배너 (Banner) | 화면 상단/하단에 고정 노출 |
| 전면 배너 (Interstitial) | 화면 전체를 덮는 팝업형 광고 |
| 네이티브 (Native) | 앱 UI에 자연스럽게 통합되는 광고 |
| 리워드 동영상 (Reward Video) | 시청 완료 시 리워드 지급 |
| 동영상 (Video) | 인라인/전면 동영상 광고 |
| 비즈보드 (Bizboard) | Kakao Adfit 비즈보드 특화 지면 |

## 지원 네트워크 (미디에이션)

- Google AdManager (GAM)
- Kakao AdFit
- Pangle
- AppLovin
- Unity Ads
- Mobwith

## 사전 준비

연동 전 [nap ssp 파트너 사이트](https://publisher.admixer.co.kr/signin)에서 미디어 등록 및 애드유닛 생성을 완료하여 **Media Key**와 **AdUnit ID**를 발급받아야 합니다.

> Google App ID, Pangle App ID 등 별도 발급이 필요한 네트워크는 운영팀(nap_adx@nasmedia.co.kr)으로 문의하세요.
