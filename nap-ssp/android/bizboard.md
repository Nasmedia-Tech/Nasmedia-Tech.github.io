## 비즈보드 지면 정책

- 비즈보드 지면은 **비즈보드만 단독 사용** (타 네트워크 미디에이션 불가)
- **지면 심사 필수**: 비즈보드 테스트 광고가 노출된 지면 스크린샷을 운영팀에 전달

## 상품 설명

앱 사용자에게 최적화된 디자인으로 맞춤형 광고를 제공하는 Kakao 프리미엄 광고 상품입니다. 철저한 심사를 거쳐 고품질 소재를 보장합니다.

## SDK 연동

Kakao Adfit Android 공식 가이드를 참고하세요:

- [비즈보드 Android 연동 가이드](https://github.com/adfit/adfit-android-sdk/blob/master/docs/BIZBOARDAD.md)

## 코드 발급 및 리포트 매핑

1. 운영팀에 비즈보드 코드 발급 요청
2. **nap ssp 파트너 사이트** → 애드유닛 이름에 '비즈보드' 입력, 포맷/사이즈: Banner - 320x50 선택
3. 발급된 애드유닛으로 운영팀에 비즈보드 리포트 매핑 요청

## 지면 심사 프로세스

```
SDK 연동 → 테스트 광고 확인 → 스크린샷 전달 → 운영팀 심사 → 상용 광고 송출
```

> 비즈보드는 반드시 심사를 거쳐야 상용 광고가 노출됩니다.
